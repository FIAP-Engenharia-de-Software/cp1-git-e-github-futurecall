# 📱 SocialConnect - FutureCall

## 📝 Resumo do Projeto
O **SocialConnect** é uma solução desenvolvida para otimizar a interação e o fluxo de dados em sistemas de comunicação. O projeto foca em oferecer uma interface via terminal (CLI) que simula o gerenciamento de conexões sociais e chamadas de suporte, priorizando uma estrutura de código limpa, escalável e de fácil manutenção, seguindo os padrões de Engenharia de Software.

---

## 🛠️ Tecnologias Utilizadas
*   **Linguagem:** HTML e CSS
*   **Versionamento:** Git & GitHub 
*   **Ambiente:** VS Code / Terminal
